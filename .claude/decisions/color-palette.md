# Decisión: Paleta de Colores

**Estado**: Aprobada
**Fecha**: Febrero 2026

## Decisión

4 colores únicos con distribución fija.

## Colores

| Color | Hex | Distribución | Razón |
|-------|-----|-------------|-------|
| **Negro Carbón** | `#1D1D1B` | 12% | No negro puro → calidez sutil, evita contraste duro |
| **Rojo Estudio** | `#B81716` | 3% | Profundo, serio, no agresivo → uso quirúrgico como firma |
| **Gris Piedra** | `#EBEBEB` | 15% | Remite a hormigón, piedra → conexión directa con arquitectura |
| **Blanco** | `#FFFFFF` | 70% | Silencio visual, espacio, respiración |

## Distribución Cromática

```
70% Blanco — fondo predominante, aire, espacio
15% Piedra — secciones alternativas, separadores
12% Carbón — textos, secciones oscuras, navbar, footer
 3% Rojo  — CTAs, líneas decorativas, hover states
```

## Principio

El rojo no grita, puntúa. Es una firma a mano en un documento impecable: pequeño, preciso, decisivo.
